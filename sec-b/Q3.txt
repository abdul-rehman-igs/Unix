Q3: Execute the following tasks
a) Use the find command to search the /usr/share directory for files that are larger than 5 MB but smaller than 10 MB.
b) Using the locate command, list files with name ending with 'sql' and display the output page by page.

perform this:

a) find /usr/share -type f -size +5M -size -10M

b) sudo updatedb
locate "*.sql" | less
